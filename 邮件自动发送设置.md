# 邮件自动发送设置

当前脚本使用 QQ 邮箱 SMTP 发送邮件：

- 发件邮箱：通过环境变量 `QQ_SMTP_USER` 设置
- 授权码：通过环境变量 `QQ_SMTP_AUTH_CODE` 设置
- 默认收件人：可通过环境变量 `DAILY_EMAIL_TO` 设置，也可运行脚本时用 `--to` 指定

## 临时测试发送

在 PowerShell 中设置本次会话环境变量后运行：

```powershell
$env:QQ_SMTP_USER="你的QQ邮箱地址"
$env:QQ_SMTP_AUTH_CODE="你的QQ邮箱SMTP授权码"
.\scripts\发送第01天测试邮件.ps1 -To "mengruogang@163.com"
```
