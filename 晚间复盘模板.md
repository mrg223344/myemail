# 生理晚间复盘模板

用途：每天 23:00 发送，用来回顾当天导图与口诀，重点做题、思考和复盘。

## 邮件结构

1. 标题
   - 生理第 XX 天晚间复盘：主题

2. 今晚复盘目标
   - 只复盘当天内容，不贪多。
   - 先闭卷回忆导图主干。
   - 再做题检查漏洞。
   - 最后写下明天开头要补的点。

3. 闭卷回忆清单
   - 5-8 个当天核心框架问题。
   - 不直接给答案。

4. 深度思考题
   - 5-8 个需要解释、比较、举例的问题。
   - 重点训练“为什么”和“如何区分”。

5. 晚间西综训练：15 道
   - 先只给题目和选项。
   - 题后单独给“答案与解析”。
   - 题目覆盖当天导图主干、口诀、易混概念。

6. 复盘记录
   - 今日最稳的 3 个点。
   - 今日最虚的 3 个点。
   - 明天早上先复盘的 3 个关键词。
   - 给今天学习状态打分：1-5 分。
