### 一、口腔内消化和吞咽
1. 唾液的分泌
- (1) 唾液的性质和成分
1性质
- 无色无味,近中性(PH : 6.6~7.1)
- 低渗液 (汗液也是)
2成分
- 水 (99%)
### 有机物
### 黏蛋白
- 免疫球蛋白 (Ig)
### 溶菌酶
- 唾液淀粉酶 ( 最适 PH :中性 )
### 尿素
- 杀菌作用→干燥综合征 (ss) →唾液缺
### 乏→猖獗性龋齿
- 无机物: Na+ 、 K+ 、 Ca2+ 、 Cl-
- 气体: O2 、 N2 、 CO2 、 NH3
- (2) 调节 (100% 完全是神经调节)
1交感 N ↑:量少,固体成分多而粘稠
2副交感 N ↑:量多,固体成分少而稀薄
### 协同作用
2. 食管下括约肌 (LES)
- (1) 解剖学上并不真正存在该括约肌,而是一段功能性的高压区:比胃内压高 5~10mmHg(所以可以防止反流)
- (2) 意义:阻止胃内容物逆流入食管
- (3) 调节
1神经调节:受副交感 N( 迷走 N) 支配
- 食物刺激食管壁→迷走 N传入→迷走 N抑制性纤维传出→释放VIP/NO→ LES 舒张
- 食物入胃→迷走 N传入→迷走 N兴奋性纤维末梢传出→释放 Ach → LES 收缩
2体液调节
- LES 收缩 = 胃运动↑ = 促胃液素、胃动素
- LES 舒张 = 胃运动↓ = 促胰液素、缩胆囊素 (CCK) 、前列腺素 A2
### 二、胃内消化
2. 胃的运动
- (1) 胃的运动形式
1紧张性收缩
2蠕动
3胃荣受性舒张
- ( 胃特有 )
- 以头区 ( 胃体 / 胃低 ) 为主
- 机制 : 进食刺激口腔、咽、食管处感受器→传入 N →中枢→迷走 N 传出 ( 抑

- (2) 胃排空及其控制
1概念:食物由胃排入十二指肠的过程称为胃排空。食物入胃后 5 分钟就开始胃排空
2排空速度
- 液体食物>固体食物,小颗粒食物>大块食物,等渗液体>非等渗液体
2三大营养物质:糖类(最快)>蛋白质>脂肪(最慢)
3影响因素
- 三、大肠的消化 1. 大肠内细菌的作用
- (1) 对糖及脂肪的分解称为发酵,产物有乳酸、乙酸、 CO2 、甲烷等
- (2) 对蛋白质的分解称为腐败,产物有氨基酸、 NH3 、 H2S 、组胺、吲哚等还能利用肠
- 内较为简单的物质来合成 Vit B 和 Vit K,这些维生素可被人体吸收利用
