### 一、消化生理概述
1. 一般生理特性
- (1)兴奋性较低,收缩缓慢:收缩潜伏期、收缩期和舒张期较长,变异较大
- (2) 自律性较低,且不规则
- (3) 保持紧张性
1保持消化道形状、位置
2保持基础压力,有利于消化液向食物渗透
3是保持收缩活动的基础
- (4) 富有伸展性:适应接纳食物的需要进行很大的伸展,以增加其容积
- (5) 对不同刺激的敏感性不同
1对电刺激不敏感( 针刺、刀割刺激不敏感)
2对机械牵拉、温度和化学刺激 (腹膜炎) 特别敏感
2. 电生理特性 ( 膜电位 )
1.RP : -50~60mv 较小、不稳定、易波动
- K+ 外流 ( 主要 )
- Cl- 、 Ca2+ 、钠泵
### 较小的原因
2. 慢波电位 ( 局部电位、基本电节律 )
- (1) 在 RP 的基础上 ,自发地产生轻度
- 去极化和复极化 , 频率慢 ( 故称慢波 )
- (2) 消化道不同部位频率不同
- 胃: 3 次 / 分
- 十二次直肠: 12 次 / 分
- 回肠末端: 8~9 次 / 分
- (3) 幅度: 10~15mv, 持续数秒 ~ 十几秒
- (4) 起源于纵行肌与环形肌之间的 Cajal 间质 c
- (5) 机制 ( 尚不清 ) :教材写尚不清的都不会考
- (6) 阈电位
- 机械阈→小幅度 ( 轻度 ) 收缩 ( 强度与慢波幅度正相关 )
- 电阈→ A[ → Ca2+ 大量内流→明显收缩 ( 强度与AP 频率正相关 )
### 三、胃肠激素
1.APUD 细胞 : 具有摄取胺的前体、进行脱羧而产生肽类或活性胺的能力。
2. 胃肠激素 : 一些内分泌细胞合成和释放的多种激素主要在消化道内发挥
- 作用,因此把这些激素合称为胃肠激素
3. 分类
4. 生理作用
### 二、消化道的神经支配
### 小红书 @ 小花生西综笔记
- 微信公众号:一颗小花生笔记
- 基本电节律:慢波频率对
### 平滑肌收缩气决定性作用
- 上游促进下游,下游抑制上游
- 胃肠激素作用:
1. 调节消化腺分泌和消化道运动
2. 调节其他激素释放
3. 营养作用:促进消化系统组织生长
