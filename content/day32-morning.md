### 听觉和平衡觉
- 听觉(一)
1. 听阈和听域
- (1) 听阈
1人耳能够感受的声波频率范围是 20~20000 Hz
2人耳最敏感的声波频率在1000 ~3000 Hz
3对每种频率的声波,都有一个刚能引起听
- 觉的最小强度,称为听阈( 右图中下方曲线 )
4在听阈以上继续增加强度,听觉的感受也相应增
- 强,当强度增加到某一限度时,将引起鼓膜的疼痛
- 感觉,这一限度称为最大可听阈(右图中上方曲线)
- (2) 听域:听阈与最大可听阈两条曲线所包绕的面积称为听域
2. 外耳和中耳的功能
- (1) 外耳的功能 :传音+增强声压
- (2) 中耳的功能 :增压减幅作用( 增强
- 声压 22.4 倍,减小振幅 1/4)
1中耳声音传播 : 鼓膜(产生振动)→听骨链(传递振动)→卵圆窗膜(内耳)
2增强声压的结构包括 :外耳道、鼓膜、听骨链、卵圆窗膜
3咽鼓管 : 是连接鼓室和鼻咽部的管道
- a. 鼻咽部开口常闭→吞咽、打哈欠时开放→空气进入鼓室→
- 平衡鼓膜内外两侧压力 (飞机起飞乘务人员要求咽口水)
- b. 咽鼓管阻塞 ( 中耳炎 ) →外界空气不能进入鼓室→鼓室内原有空气
- 被吸收→鼓室内压力下降→鼓膜内陷→鼓膜疼痛、听力下降、耳闷
4中耳鼓室鼓膜张肌和镫骨肌作用 :对内耳的感音装置起到保护作用 ,
- 但需要 40~160 毫秒,故对突发性爆炸声的保护作用不大 .
3. 声波传入内耳的途径
- (1) 气传导 (2 条 ): 声波→外耳道→鼓膜
1→听骨链→卵圆窗膜→前庭阶( 外 ) →基底膜
2→鼓室空气→圆窗膜→鼓阶( 外 ) →基底膜
- ( 听骨链正常时并不重要 , 仅在听骨链损害时起作用 , 但听觉敏感度大为降低 )
- (2) 骨传导 (1 条 ): 声波→颅骨→耳蜗→基底膜( 没有经过外耳、中耳 )
- ( 也不重要 , 只有在听骨链损害时起作用 , 听觉敏感度大为降低 )
- (3) 耳聋
1传音性耳聋 : 声音传导障碍 —部位 : 外耳 + 中耳
2感音性耳聋 : 声音感受障碍 —部位 : 内耳 ( 螺旋器、血管纹 )
### 平衡觉
1. 半规管
2. 椭圆囊和球囊
- 感受器是囊斑,其适宜刺激是直线加速度运动
- (1) 当人体直立而静止不动时,椭圆囊囊斑的平面与地面平行,感受水平方向的直线加速度( 乘地铁 )
- (2) 当人体直立而静止不动时,球囊囊斑的平面则与地面垂直,感受垂直方向的直线加速度( 乘电梯 )
3. 眼震颤 ( 前庭反应 )
- 是指身体在旋转变速运动时 ( 感受刺激: 半规管) 出现的眼球不自主的节律性运动
- (1) 两侧水平半规管受到刺激时可引起水平方向的眼震颤
- (2) 上半规管受刺激时可引起垂直方向的眼震颤
- (3) 后半规管受刺激时可引起旋转性眼震颤
- 听觉(二)
4. 耳蜗的感音换能作用
- (1) 基底膜的振动 : 声波振动→听骨链→卵圆窗膜→
1当卵圆窗膜内移时→前庭膜和基底膜下移→鼓阶的外淋
### 巴液压迫圆窗膜→圆窗膜外移
2当卵圆窗膜外移时→前庭膜和基底膜上移→鼓阶的外淋
### 巴液内移回撤退→圆窗膜内移
- 如此反复 , 形成基膜振动
- (2) 行波理论
1不同频率的声波引起的行波都是从基底膜底部( 靠近卵圆窗膜处 ) 开始
2行波的起点与终点之间有一个振幅最大的部位
3声波频率不同 , 行波传播的距离
### 和最大振幅出现的部位有所不同
1声波频率越高→行波传播越近→最大振幅出现的部位越靠近蜗底
2声波频率越低→行波传播越远→最大振幅出现的部位越靠近蜗顶
- 高调的人走不远 , 在底部
- 低调的人行千里 , 在顶峰
- (3) 耳蜗 ( 内耳 ) 的感音换能机制
1内毛细胞产生感受器电位→细胞基底侧膜上电压门控 Ca2+ 开放→ Ca2+ 内流→细胞内
- Ca2+ ↑→触发细胞底部释放神经递质→听神经 / 前庭神经上 AP →中枢
2外毛细胞马达( Prestin )蛋白→去极化使胞体缩短→ 基底膜振动(耳蜗放大器效应,使
- 内毛细胞更易受到刺激→敏感性↑)
3感音换能过程
- 感音 : 声波→外耳道→鼓膜→听骨链→卵圆窗膜→前庭阶 ( 外 ) →前庭膜→我管 ( 内 ) →基底膜 ( 螺旋器 )
- 换能 : 毛 c 听毛弯曲→机械门控 K+ 通道开放→内淋巴 (高 K+ 环境) 中 K+顺浓度进入毛 c( 去极化 ) →毛 c 去极化感受器
- 电位 (局部电位) →毛 c 基底部电压 Ca2+ 激活→ Ca2+ 内流触发囊泡递质释放→听神经爆发 AP→听觉中枢产生听觉
5. 耳蜗的生物电现象
- (1) 耳蜗内电位
1毛细胞静息电位- 70~ - 80mV
2蜗管内是高 K+ 环境→耳蜗内电位是正电位→ +80 mV
3产生和维持蜗管内是高 K+ 环境与蜗管外侧壁血管纹活动有关
- 血管纹结构钠钾泵、 Na+—K+-2Cl- 同向转运体、缝隙连接、钾通道
### 影响因素
- ◆缺氧、哇巴因→抑制钠钾泵→蜗管(内淋巴)高钾环境不能维持→感音性耳聋
- ◆长期大剂量使用呋塞米和依他尼酸→抑制Na+—K+-2Cl-同向转运体→蜗管
- (内淋巴)高钾环境不能维持→感音性耳聋
- (2) 耳蜗微音器电位
1等级式反应:即其电位随着刺激强度的↑而↑( 可发生叠加总和 )
2 “ 四不 ”:无阈值、没有潜伏期和不应期、不易疲劳、不发生适应现象
