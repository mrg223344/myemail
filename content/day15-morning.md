### 三、心肌的生理特性
### 兴奋性
- (1) 周期性变化
- (2) 心室肌有效不应期很长的原因
- IK1 的内向整流特性→膜去极化时 GK1 ↓→ K+ 在心室肌 AP 2 期外流↓→复极化时间长 (2
- 期形成平台期 ) →心室肌 AP 时程长 ( 较骨骼肌 ) →有效不应期很长→一直持续到舒张早期→
### 心室肌不发生完全强直收缩
- (3) 在相对不应期和超常期爆发 AP 的特点
- ( 慢、小、低、短 )
### 相对不应期钠通道还没
### 完全恢复到静息水平
1 0 期去极化速度慢、幅度小→兴奋传导速度慢
2时程短→不应期短→易出现期前兴奋(早搏)
3兴奋性:相对不应期低,超常期高(特殊点)
- (4) 影响兴奋性的因素
1静息电位或最大复极电位水平
2阈电位水平
- a. 电压门控 Na 通道 /Ca 通道的分布密度
- b. 电压门控 Na 通道 /Ca 通道的功能状态
- c. 细胞外 Ca2+ 水平
- (5) 期前兴奋和期前收缩
1如果在心室肌的有效不应期后,下一次窦房结兴奋到达前,心室受到一次外来刺激,则可提前产生
- 一次兴奋和收缩,分别称为期前兴奋和期前收缩
2正常下传的窦房结兴奋落在期前兴奋的有效不应期内→在期
- 前收缩后会出现一段较长的心室舒张期,称为代偿间歇
### 房性期前收缩→不完全代偿间歇
### 室性期前收缩→完全代偿间歇
### 传导性
- (1) 传导系统:窦房结→心房肌→房室交界(房室结)→房室束、左右束支→浦肯野纤维→心室肌
- (2) 传导方式:通过局部电流的形式经心肌细胞闰盘(缝隙连接)在细胞之间迅速传播,引起所有细胞几乎同步兴奋和收缩
- (3) 传导速度:传导速度最慢:房室交界( 0.02m/s )(原因:非常纤细且是慢反应,薄弱点)
- 传导速度最快:浦肯野纤维( 4m/s )(原因:浦肯野纤维直径十分粗大)Vs 浦肯野自律性最低
- (4) 影响因素
1结构因素:直径越大→传导速度越快
2动作电位 0 期去极化速度和幅度 (最重要,速度越快、幅度越大则传导速度越快 )
3邻旁未兴奋区心肌膜的兴奋性 ( 邻近细胞兴奋性低→则很小的局部电流即可使临近细胞兴奋→完成 AP 的传导 )
- 房室延搁:心肌兴奋在房室交界处传导速度最慢,因此兴奋经过房室此处将出现一个时间延搁,称为房室延搁
### 自律性
- (1) 自律性高低 窦房结最高 (100 次 / 分 , 但收迷走神经影响 , 自律性为 70 次 / 分 ) >房室结 (50 次 / 分 ) >末梢浦肯野 (25 次 / 分 )
- (2) 窦房结是心脏活动的正常起搏点,窦房结起搏而形成的心脏节律称为窦性节律其他自律组织为潜在起搏点,
- 正常仅起兴奋传导作用,而不表现自身节律性
- (3) 窦房结控制潜在起搏点的主要机制:抢先占领(夺获)和超速驱动压抑
- (4) 影响自律性的因素
1 4 期自动去极化速率 ( 最重要 )
2最大复极电位 (MRP) 水平
3阈电位水平
1衡量心肌自动节律性高低的指标是自动兴奋频率
2衡量心肌自动节律性高低的最主要指标是 4 期膜电位自动去极化速率
### 收缩性
- (1) 同步收缩(全或无式收缩 / 功能合胞体)
- (2) 不发生完全强直收缩
- (3) 对细胞外 Ca2+ 依赖性高
### 一、心肌细胞的分类
- 根据 4 期有无自动去极化
### 工作细胞
### 心室肌细胞
### 心房肌细胞
### 无自律性
- 自律细胞:有自律性,可自发产生 AP
- 根据 0 期自动去极化快慢
- 快反应细胞:快钠通道 (INa) 开放产生 AP去极化速度快、幅度大→兴奋传导快
- 慢反应细胞:慢钙通道 (ICa-L) 开放产生 AP去极化速度慢、幅度小→兴奋传导慢
### 二、心肌细胞的跨膜电位
### 心室肌细胞
- (1) 静息电位
1大小:约- 90 mV
2产生机制: K+ 经心肌细胞膜上的内向整流钾通道( IK1 )外流达平衡后
- 产生( RP≈EK )
- (2) 动作电位
### 窦房结 P 细胞
- (1) 最大复极电位由于窦房结 P 细胞膜上 IK1 通道较少,因此最大复极化电位较
- (约- 70 mV )
- (2) 动作电位
### 浦肯野细胞
- (1)AP 形状与心室肌相似,但存在 4 期自动去极化 ( 自律细胞 ), 0~3 期的产生机制与心室肌
- 细胞基本相同 ( 在所有心肌细胞中,浦肯野细胞的 AP 时程最长)
- (2) 最大复极电位:较心室肌细胞 RP 更负 (接近- 100 mV ,原因:膜上 IK1 通道密度较高)
- (3)4 期 ( 自动去极化期 )
1外向: IK 进行性衰减
2内向: If 进行性增强 (主要 , 起搏电流)
- (4) 浦肯野细胞 If 通道密度很低+ 激活开放速度较慢(慢 Na+ 通道)→ 4 期自动去极化速
- 率很慢( 0.02V/s )→自律性最低 (25 次 / 分 )
### 因此
1在正常窦性心律条件下,浦肯野细胞的节律性活动受到来自窦房结的超
- 速驱动压抑( 不会体现出自己的自律性 )
2窦性节律一旦停止,浦肯野细胞的自发节律也不能立即发生,而是需要
- 一定的时间后才能开始。这也就是在III度(完全性)房室传导阻滞突然发
- 生时,心室在一个时期内停搏的主要机制
### 与 Na 泵活性增强有关
- ( 生理书 p108)
- 心房肌 / 心室肌 / 浦肯野
- 窦房结 / 房室结 ( 除快反应细胞三个外都是 )
- 生理意义:可使心室的收缩必定发生在
- 心房的收缩完毕之后,因而可避免房室
- 的收缩重叠(避免心房心室同时收缩→
- 无全心收缩期)→心房收缩在前、心室
### 在后→有利于心室充盈及射血
