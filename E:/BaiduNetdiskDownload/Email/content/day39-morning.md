- 糖皮质激素( GC )
1. 对物质代谢的影响
- (1) 糖代谢 :血糖↑ 机制 : 组织对糖的利用↓, 肝糖异生↑
- (2) 脂肪代谢 不同部位脂肪酶对 GC 敏感性不同→脂肪重新分
- 布→促进四肢脂肪分解,向躯干转移(堆积)
- (3) 蛋白质代谢
1抑制肝外组织细胞内的蛋白质合成,加速其分解
2促进肝细胞内蛋白质的合成,使肝内蛋白质和血浆蛋白增加
- Cushing 综合征:
1尿糖 (+) → Gc 对糖代谢影响
2满月脸水牛背向心性肥胖→ Gc 对脂代谢影响
3皮肤紫纹、肌肉消瘦、骨质疏松、皮肤变薄
### → Gc 对蛋白质代谢影响
2. 对血细胞的影响
- (1) 使红细胞、血小板(治疗: ITP)和中性粒细胞的数量增加 (三系↑)
- (2) 使淋巴细胞(长期应用 GC →免疫功能下降,容易发生感染)和嗜酸性粒细胞数量↓( Gc 可治疗自身免疫性疾病)
3. 对胃肠道的影响GC →PG ↓→胃酸和胃蛋白酶原↑(去抑制效应)→长期大量应用 GC/ 烧伤 / 脑外伤易诱发或加重 PU(应激性溃疡)
4. 对循环系统的影响
- (1) 对儿茶酚胺的允许作用
1血管平滑肌(α 1R ) / 心肌(β 1R ):上调受体 + 亲和力↑→强心 + 收缩血管→维持正常血压
2气管平滑肌(β 2R ):气道舒张→气道阻力↓ ( GC 可治疗支气管哮喘)
- (2) 维持循环血量
- GC → PG ↓→毛细血管通透性↓→血浆滤过↓→维持循环血量 ( GC 治疗休克)
- 同时 PG(内源性致热原) ↓→退热( GC 能抑制产热,临床上可用于发热患者的退热)
- (3) 调节水盐代谢
1保 Na+ 、排水、排 K+
2减少小肠黏膜吸收钙,抑制肾重吸收钙和磷,增加其排泄量( Cushing 综合征:骨蛋白质↓ + 血钙↓→骨质疏松)
6. 对神经系统的影响GC 能维持 CNS 的正常兴奋性,改变行为和认知能力,过量 GC 可引起失眠、情绪激动或压抑,
- 记忆力减退等症状( Cushing 综合征:出现精神异常)
7. 参与应激反应
- (1) 应激反应和应急反应
- (2) 应激时的激素变化
1应激时升高的物质: CRH 、 ACTH 、 GC 、儿茶酚胺、生长激素、催乳
- 素、 ADH ( VP )、β - 内啡肽(镇痛作用)、胰高血糖素、醛固酮等
2应激时降低的物质:甲状腺轴所有激素( TRH 、 TSH 、甲状腺激素)、性腺轴
- 所有激素(如 GnRH 、 LH 、 FSH )和胰岛素(交感神经兴奋抑制胰岛素分泌)
- 记少不记多,记减少即可

- (1) 由胰岛β( B )细胞分泌,是唯一降血糖的激素
- (2) 生物作用 :减少来源,增加去路→血糖↓,能促进生长,但不能促进产热
### 糖的来源和去路
- (3) 作用机制
- (4) 分泌调节
2. 胰高血糖素
- (1) 由胰岛 A (α)细胞分泌
- (2) 生物作用↑来源,↓去路→血糖↑
- (3) 靶器官 : 肝脏
- (4) 信号转导通路: GPCR-PKA 通路→肝糖原分解 + 肝糖原合成
