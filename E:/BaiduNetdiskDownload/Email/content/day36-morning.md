### 躯体运动的调控
### 脊髓对躯体运动的调控作用
1. 脊休克
- (1) 脊休克的产生与恢复,说明脊髓具有完成某些简单反射的能力 ( 脊髓是躯体和
- 内脏反射的初级中枢 ),但这些反射平时受高位中枢的控制而不易表现出来
- (2) 脊休克产生的机制:离断面以下脊髓突然失去高位中枢 ( 大脑皮层、脑干网状
- 结构等 ) 的调控,而不是由于切断脊髓时的损伤刺激所引起
- (3) 脊休克的恢复
2. 脊髓前角运动神经元
- (1) α运动神经元
1既接受来自脑干到大脑皮层各级高位运动中枢的下传信息
2又接受接受来自躯干、四肢皮肤、肌肉和关节感受器的外周信息传入
3α运动 N 元是躯体运动反射的最后公路
4α运动 N 元 ( 阵发性放电 ) → A α→ ACh →直接导致骨骼肌中的梭外肌收缩
- (2) γ运动 N 元
1只接受来自大脑皮层和脑干等高位中枢的下行调控
2γ运动 N 元 (持续性放电,兴奋性较α高 ) → A γ→ ACh →
- 骨骼肌中的梭内肌收缩 ( 调节肌梭对牵拉刺激的敏感性)
3. 运动单位
- (1) 定义 : 由一个α运动 N 元及其所支配的全部肌纤维所组成的功能单位
- (2) 运动单位的大小相差很大,其大小取决于α运动 N 元轴突末梢分支的多少
1运动单位大→产生肌张力大→使许多肌纤维同时发生收缩 ( 空间总和 )
2运动单位小→利于肌肉的精巧运动
4. 脊髓对姿势反射的调节
- (1) 定义:中枢神经系统通过反射改变骨骼肌的肌紧张或产生相应的动作,以保持或改变身体的姿势
- 避免发生倾倒,称为姿势反射
- (2) 意义:保持平衡,以防跌倒。
- (3) 姿势反射包括
1对侧伸肌反射一侧肢体受到伤害性刺激
- 首先同侧肢体发生屈曲(屈肌反射) →意义 :躲避伤害
- 若刺激强度↑→对侧肢体伸展( 对侧伸肌反射 ) →意义 :保持平衡
2牵张反射
3节间反射等等脊休克恢复的后期可出现较复杂的节间反射
### 大脑皮层对躯体运动的调控
1. 运动传出通路损伤时的表现
2. 巴宾斯基征( Babinski 征)
- (1) 高位中枢平时具有抑制屈肌反射(原始反射)的作用,因此表现不明显(微弱)
- (2) 皮层脊髓束(高位中枢)损伤→屈肌反射被释放而明显增强→表现为拇指背(伸),
- 其余四趾扇形展开(异常的跖伸肌反射,背伸 = 背屈,即屈肌反射)→躲避伤害
- (3) 假阳性:婴儿(皮层脊髓束发育不完全→高位中枢功能↓),成人深睡或麻醉情况下(高位中
- 枢功能↓)。因此查体 Babinski 征阳性提示皮层脊髓侧束损伤的前提条件成人在清醒状态下

### 基底神经节对躯体运动调控
1. 基底 ( 神经 ) 节
- (1) 新纹状体:尾核 + 壳核
- (2) 旧新纹状体:苍白球
### 中脑黑质 + 丘脑底核 +
1. 帕金森病的主要症状中
1静止性震颤与丘脑外侧腹核功能异常有关→左旋多巴和 M 受
### 体拮抗剂治疗无效
2其余与多巴胺系统受损和乙酰胆碱系统亢进有关→左旋多巴
### 和 M 受体拮抗剂治疗有效
2.帕金森病和舞蹈病分别是由于黑质和新纹状体病变所致 —— 记为 “文武双全还怕黑”
### 脑干对肌紧张和姿势的调控
1. 去大脑僵直现象 : 在麻醉动物,于中脑上、下丘之间切断脑干→中断了大脑皮层、纹状体等
- 部位与脑干网状结构之间的功能联系→抑制区↓和易化区↑→肌紧张明显亢进,表现为四肢
- 伸直、坚硬如柱、头尾昂起、脊柱挺硬、呈角弓反张状态
1. 切断腰骶部后根消除肌梭传入→去大脑僵直现象消失→经典的去大脑僵直属于γ僵直
### 小红书 @ 小花生西综笔记
- 微信公众号:一颗小花生笔记
- 依据:脊休克恢复后的动物,在第一次离
- 断的水平下方再做第二次脊髓离断手术,
- 离断水平以下的部位不再发生脊休克现象
### VS 屈肌反射
1不是姿势反射
2脊休克恢复后增强( 原因:高位中枢平时对屈肌反射是抑制的 )
3 Babinski 阳性(背屈) / 阴性(跖屈)都是屈肌反射
4多突触反射
5后发放 / 后放电现象 ( 环式联系 )
- 第五颈髓( C5 )水平以下切断脊髓。脊
- 髓与高位中枢离断后,暂时丧失反射活
- 动能力进入无反应状态称为脊休克
