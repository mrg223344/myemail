### 心输出量
- 每搏输出量(搏出量)
- 定义:一侧心室一次心脏搏动所射出的血液量
- 计算:舒张末期容积(原来)-收缩末期容积
- (剩下) =125ml - 55ml=70ml ( 60~80ml )
- 射血分数( EF )
- 射血分数=搏出量 / 心室舒张末期容积[左室射血分
- 数( LVEF )正常值:55%~65%]
- 心室功能减退、心室异常扩大的患者:扩心病每搏量基本不变但 EF 明显↓
- 每分输出量(心输出量 / 心排出量, CO )
- 定义:一侧心室每分钟射出的血液量
- 不能作为不同身材个体心功能评价指标
- 心脏指数( CI )
- 以单位体表面积计算的心输出量,心指数=心输出量 / 体表面积
- 心指数可用于不同身材的个体心功能的评价

### 影响心输出量的因素
### 前负荷 = 舒张末容积
1心功能曲线不会有明显下降支
- 原因 a :正常心室肌具有抗过度延伸的特性,伸展性较小
- 原因 b :心肌肌节内存在连接蛋白,可限制肌节的被动拉长
2使用肾上腺素→胞质内 Ca2+ ↑→心肌收缩能力↑→心功能曲线向左上移动
- 后负荷 = 大动脉血压动脉血压 / 后负荷↑
1半月瓣开放延迟
2等容收缩期延长、射血期缩短、射血速度下降
3射血时最高室压(峰值)升高、达峰时间延长
### 每搏量↓←
- 心肌收缩能力(等长调节)
1儿茶酚胺→激动β 1 受体→ GPRCR → PKA 使 L 型钙通道磷酸化激活→ Ca2+ 内流增加→心肌收缩力↑
2钙增敏剂(如茶碱 / 左西孟旦)增加肌钙蛋白对 Ca2+ 的亲和力→肌钙蛋白对胞质中 Ca2+ 的利用
- 率↑→活化的横桥数目↑→心肌收缩能力↑
3甲状腺激素→提高肌球蛋白 ATP 酶的活性→心肌收缩能力↑→每搏量↑
### 心率
1在一定范围内,心率↑→心输出量↑
2心率过快(心动过速)> 160~180 次 / 分→心动周期↓(舒张期明显缩短)
- →心室充盈时间明显↓→心室充盈量明显↓→每搏量明显↓→心输出量↓
3心率过慢< 40 次 / 分→心动周期↑:舒张期过长→心室充盈早已接近最大限
- 度,心舒期的延长已不能进一步增加充盈量和每搏量→心输出量↓

### 影响肌肉收缩效能的因素
- ( 原第二章内容)
- (1) 外在因素(负荷)
1前负荷( = 初长度)
- ◆在一定范围内增加前负荷↑肌肉收缩张力↑
- ◆最适初长度 = 产生最大收缩张力的初长度 = 肌节长度为 2.0~2.2 μ m= 全部横桥都能发挥作用
- ◆肌肉一般都处于最适初长度状态,以利于产生最大的收缩张力
- ◆影响机制:调节粗、细肌丝间的重叠程度 / 与细肌丝(肌动蛋白)接触的横桥数目
- ◆前负荷(初长度)只影响肌肉收缩的主动张力,不影响缩短程度和速度
2后负荷
- ◆在一定范围内后负荷↑肌肉收缩张力↑
- ◆但后负荷↑→肌肉缩短速度↓,缩短程度↓,开始收缩时间延迟
- ◆影响机制:影响横桥周期长短,后负荷↑→横桥周期↑
- (2) 内在因素(肌肉收缩能力)
1胞质内 Ca2+ 浓度的改变
2与肌丝滑行有关的横桥 ATP 酶活性
3肌细胞能量代谢水平
4各种功能蛋白及其亚型的表达水平
5肌原纤维的肥大与否等
- (3) 收缩的总和(等级性收缩)
1骨骼肌收缩的空间总和 / 多纤维总和 / 多运动单位总和 多个运动神经元同时兴奋→同步收缩运动单位数目↑→收缩(张)力↑
2骨骼肌收缩的时间总和 / 频率总和
- 单收缩 = 完整的收缩 + 完整的舒张
### 强直收缩
- 不完全强直收缩 = 完整收缩期 + 不完整舒张期
- 完全强直收缩 = 不完整收缩期 + 不完整收缩期
- 初长度(前负荷)一定范围内↑→心肌收缩力↑→
- 每搏量↑ = 异长自身调节 =Frank-Starling :心室
- 舒张末期容积一定范围内增大→心肌收缩力↑
### 血压↓
1心室收缩末期容积↑→心室舒张末期容积(前负荷)↑→每搏量↑(异长调节)
2神经 - 体液(交感神经兴奋,儿茶酚胺) / 心肌收缩能力↑→每搏量↑(等长调节)
